How to run this on RunPod (6 steps)
  1. Build & push the worker image (build context = repo root): 
    docker buildx build --platform linux/amd64 \
  -f runpod/Dockerfile \
  -t YOURUSER/claude-animations-runpod:latest --push .
  2. Create a RunPod Serverless endpoint on that image — cheapest GPU is fine, Max workers ≈ 20.
  3. Set the bucket secrets on the endpoint: S3_ENDPOINT, S3_BUCKET, S3_ACCESS_KEY_ID, S3_SECRET_KEY, S3_PREFIX, PUBLIC_BASE_URL.
  4. Submit the 50 jobs from your laptop: 
    RUNPOD_API_KEY=… RUNPOD_ENDPOINT_ID=… \
  node runpod-submit.mjs   # → batch-mcp/outputs.json
  5. Point this page at your bucket — set OUTPUT_BASE_URL below, or commit batch-mcp/outputs.json.
  6. git add && commit && push — the clips go live here.
